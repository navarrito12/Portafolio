ESTRUCTURA BÁSICA : Le dice al navegador que estás usando HTML. : Envuelve todo el contenido. : Contiene información "invisible" para el usuario, como el título de la pestaña (). <body>: Aquí va todo lo que el usuario sí ve (párrafos, imágenes, listas). TIPOS DE LISTAS Listas Desordenadas (<ul>): Se usan para elementos sin un orden específico (usan "bullets" o puntos). Listas Ordenadas (<ol>): Se usan cuando el orden importa (1, 2, 3...). Elementos de lista (<li>): Es la etiqueta que envuelve cada uno de los ítems dentro de una lista, ya sea ordenada o desordenada. Para invertir el orden, se usa el atributo reversed en la etiqueta <ol>: <ol reversed>. * Para cambiar a números romanos, se usa el atributo type: <ol type="I">. ENLACES A IMÁGENES Enlaces (<a>): Se usa el atributo href para el destino. Ejemplo: <a href="http://google.es">Ir a Google</a>. Imágenes (<img>): Es una etiqueta que no se cierra. Usa src para la ruta y alt para el texto alternativo (muy importante para la accesibilidad). Ejemplo: <img src="logo.png" alt="Logo de Google"> MARCADORES INTERNOS Para saltar a una sección de la misma página, necesitas dos pasos: Poner un ID al destino: <h2 id="origen">El origen</h2>. Crear el enlace con un almohadilla (#): <a href="#origen">Ir al origen</a>.
